h1. Welcome to Blogcast

Blogcast is a better way to blog.

bq. Blogcast is a simplistic Rails 3 blogging engine which has some real nice intelligent defaults and tries to keep things simple so you can focus on writing. 
--Gregg Pollack & Nathaniel Bibler, Ruby5 Podcast

h2. Installation

<pre>
git clone git://github.com/tiandavis/blogcast.git
cd blogcast
bundle install
rake db:create
rake db:migrate
rake db:seed
</pre>

h2. Getting Started

<pre>
rails s
http://localhost:3000/admin
login with l/p: admin
</pre>
